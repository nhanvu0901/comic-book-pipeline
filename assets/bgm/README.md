# Background Music

Drop a royalty-free MP3/WAV here named `default.mp3` (or any path you configure via `BG_MUSIC_PATH` in .env).

Recommended sources (royalty-free, no attribution required for personal/YouTube use):
- Pixabay Music: https://pixabay.com/music/ (Cinematic / Inspirational / Documentary tags)
- YouTube Audio Library: https://studio.youtube.com (Audio Library)
- Free Music Archive: https://freemusicarchive.org

Stage 5 will mix the BGM at 0.35 gain with sidechain ducking against the narration (~6 dB duck during speech). If no file is found, the final video is narration-only.
